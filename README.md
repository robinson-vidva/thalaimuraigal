# Thalaimuraigal - Family Tree Website

A Next.js family tree application with TypeScript, Tailwind CSS, and Prisma (SQLite).

## Getting Started

```bash
npm install
npx prisma migrate dev
npm run db:seed
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript**
- **Tailwind CSS v3**
- **Prisma v5** with SQLite
- **Vercel** for deployment
